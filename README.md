# Arduino based Access Control System project
### Components:
- acs-BE: backend
- acs-FE: frontend
- acs-FW: firmware
- acs-DB: database
- acs-WS: web server

### Technologies and tools:
- Java EE
- Angular
- Arduino
- MySQL
- WildFly
- Node.js
- Maven
- Jenkins
- SonarQube
- IntelliJ IDEA
- Postman
- Git
